# LJPS Front-End

## Move to vue-app directory 

```
cd frontend/vue-app 
```

## Project setup (if you've already done before, pass this step)
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
